# Design Ideas - Gabriel Mateus Portfolio

## Chosen Design Direction

**Style:** Minimalista Elegante com Foco em Fotografia

### Visual Identity
- **Paleta de Cores:**
  - Background principal: Branco puro (#ffffff)
  - Background secundário: Cinza muito claro (#f8f9fa)
  - Texto principal: Cinza escuro (#1a1a1a)
  - Accent: Dourado suave (#c9a961) para detalhes e hover states
  - Overlay: Preto semi-transparente para texto sobre imagens

- **Tipografia:**
  - Títulos: Playfair Display (serif elegante)
  - Subtítulos: Montserrat (sans-serif moderna)
  - Corpo: Inter (sans-serif legível)

- **Layout:**
  - Hero fullscreen com imagem de fundo e texto centralizado
  - Grid assimétrico para galeria de fotos
  - Seções com espaçamento generoso (breathing room)
  - Imagens em destaque com aspect ratio 3:2 e 4:5
  - Navegação fixa transparente que se torna sólida no scroll

### Key Features
1. **Hero Section:** Imagem fullscreen com overlay escuro, nome do fotógrafo em tipografia grande e elegante
2. **Sobre:** Layout em duas colunas (foto + texto) com foto em preto e branco
3. **Serviços:** Cards com ícones minimalistas e hover effects suaves
4. **Portfólio:** Masonry grid com filtros por categoria, lightbox para visualização
5. **Contato:** Formulário clean com validação, informações de contato visíveis

### Interactions
- Scroll suave entre seções
- Parallax sutil no hero
- Fade-in ao scroll para elementos
- Hover effects nas imagens (zoom suave)
- Transições de 300-400ms para suavidade

### Responsiveness
- Desktop: Grid de 3-4 colunas no portfólio
- Tablet: Grid de 2 colunas
- Mobile: Coluna única, navegação hamburger
